## 🚀 快速开始

```bash
git clone https://github.com/0xRick0x/cryptoskills-all.git
cp -r cryptoskills ~/.claude/skills/

# Binance
npx skills add binance-web3

# OKX
npx skills add okx/agent-skills

# Gate
npx skills add gate/gate-skills   # 或者一键安装
```
